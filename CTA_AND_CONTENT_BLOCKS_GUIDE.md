# CTA Management & Content Blocks Guide

## How CTA Management Works

### 1. **CTA Blocks** (Reusable Components)
- **Purpose**: Create reusable call-to-action blocks that can be used across multiple blog posts
- **Types Available**:
  - Button CTA
  - Banner CTA
  - Inline Text CTA
  - Popup/Modal CTA
  - Sidebar CTA
  - Footer CTA
  - Form CTA
  - Download CTA
  - Custom HTML

- **Features**:
  - Each CTA block has a name, title, description, button text, and URL
  - Can be styled with different color schemes (primary, secondary, success, etc.)
  - Can include images for banner/popup CTAs
  - Can track conversions and clicks
  - Can be activated/deactivated

### 2. **CTA Placements** (Where CTAs Appear)
- **Purpose**: Control where and how CTAs appear in specific blog posts
- **Placement Types**:
  - Auto-insert at top
  - Auto-insert in middle
  - Auto-insert at bottom
  - After specific paragraph
  - After specific heading
  - Manually placed

- **Features**:
  - Link a CTA block to a specific blog post
  - Set position in content (paragraph/heading index)
  - Set display conditions (e.g., show after 50% scroll)
  - Track clicks and conversions per placement
  - Enable/disable placements

### 3. **Auto-Placement Feature**
- Automatically inserts up to 3 CTAs into a blog post at strategic positions:
  - Top (position 0)
  - Middle (calculated based on content length)
  - Bottom (last paragraph)

## Content Blocks System

### What Are Content Blocks?
Content blocks are reusable templates that can be inserted into blog posts to add rich, structured content sections. They make each blog post unique and information-rich.

### Available Block Types:
1. **Data Table** - Structured data tables
2. **Info Box** - Highlighted information boxes
3. **Warning Box** - Warning/alert boxes
4. **Tip Box** - Helpful tips and hints
5. **Quote Block** - Pull quotes and testimonials
6. **Statistics Block** - Display key statistics
7. **Timeline** - Chronological timelines
8. **Comparison Table** - Side-by-side comparisons
9. **Testimonial Block** - Customer testimonials
10. **Pricing Table** - Pricing information

### How It Works:
1. **Create Templates**: Create reusable content block templates with HTML/template content
2. **Insert into Blogs**: Link templates to specific blog posts at specific positions
3. **Customize**: Override template data with blog-specific custom data
4. **Auto-Insert**: Some blocks can be auto-inserted (e.g., table of contents)

### Benefits:
- **Consistency**: Reusable templates ensure consistent styling
- **Efficiency**: Create once, use many times
- **Rich Content**: Add tables, info boxes, timelines, etc. without coding
- **Flexibility**: Customize each instance with blog-specific data
- **SEO**: Structured content blocks improve SEO
