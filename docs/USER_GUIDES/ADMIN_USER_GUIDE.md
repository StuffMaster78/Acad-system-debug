## ‚ùì FAQ

### User Management

**Q: How do I suspend a user?**  
A: Go to Users, find the user, click "Suspend", add reason, and confirm.

**Q: Can I restore a deleted user?**  
A: Yes, use the restore function in user management.

### Orders

**Q: How do I manually assign a writer?**  
A: Open the order, go to Assignment, select writer, and confirm.

**Q: Can I modify an order after payment?**  
A: Yes, but changes may affect pricing. Contact support for complex changes.

### Disputes

**Q: How long do I have to resolve a dispute?**  
A: Disputes should be resolved within 48 hours. Check SLA settings.

**Q: Can I partially resolve a dispute?**  
A: Yes, you can create partial resolutions with specific conditions.
