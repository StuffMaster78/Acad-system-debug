## üöÄ Getting Started

### First Login

1. **Access the System**
   - Navigate to the editor portal

   - Enter your email address

## üìù Task Management

### Viewing Available Tasks

1. **Access Task Queue**
   - Go to **"Tasks"** ‚Üí **"Available Tasks"**
   - See tasks you can take

2. **Task Information**
   - Order details
   - Writer information
   - Submission date
   - Priority level
   - Estimated time

### Taking Tasks

1. **Review Task Details**
   - Click on a task to view full details
   - Review submitted work
   - Check requirements

2. **Take Task**
   - Click **"Take Task"** button
   - Confirm you can complete on time
   - Task will be assigned to you

### Managing Assigned Tasks

#### Task Statuses

- **In Progress** - Currently editing
- **On Hold** - Temporarily paused
- **Under Review** - Reviewing work
- **Completed** - Task finished
- **Returned** - Returned to writer

#### Task Actions

**Start Editing**
1. Open the task
2. Review submitted work
3. Click **"Start Editing"**
4. Begin quality review

**Submit Review**
1. Complete editing
2. Add review notes
3. Select approval status
4. Click **"Submit Review"**

**Return to Writer**
1. Open the task
2. Click **"Return to Writer"**
3. Provide feedback
4. Submit return

## üîß Troubleshooting

### Common Issues

#### Can't Take Task
- **Check**: Are you at capacity?
- **Check**: Is your account active?
- **Solution**: Complete current tasks or contact support

#### Task Not Loading
- **Check**: Internet connection
- **Check**: Browser compatibility
- **Solution**: Refresh page or contact support

#### Can't Submit Review
- **Check**: All required fields filled?
- **Check**: Files uploaded?
- **Solution**: Complete all required steps

## ‚ùì FAQ

### Tasks

**Q: How many tasks can I take?**  
A: Depends on your capacity. Check your workload management dashboard.

**Q: What if I can't complete a task on time?**  
A: Contact support as soon as possible to discuss options.

**Q: Can I return a task to the writer?**  
A: Yes, use the "Return to Writer" option with feedback.

### Performance

**Q: How is my performance measured?**  
A: Based on completion rate, quality score, and client satisfaction.

**Q: Can I see my performance trends?**  
A: Yes, check the Performance dashboard for trends over time.
