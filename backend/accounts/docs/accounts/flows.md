# Accounts Flows

## Registration Flow

1. User registers (authentication)
2. User created (users)
3. AccountProfile created (accounts)
4. Default role assigned (client)
5. Onboarding triggered
6. Audit log recorded

---

## Role Assignment Flow

1. Admin triggers assignment
2. Request validated
3. Role resolved
4. Role assigned
5. Audit log written

---

## Account Suspension Flow

1. Admin triggers suspension
2. Status transition validated
3. Status updated
4. Suspension fields updated
5. Audit log written
